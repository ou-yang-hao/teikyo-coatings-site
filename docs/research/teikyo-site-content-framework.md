# 湖南帝京新材料英文官网内容框架

- 研究对象：[https://en.teikyo.com.cn/](https://en.teikyo.com.cn/)
- 原始入口：[News Center](https://en.teikyo.com.cn/news/1/)
- 访问日期：2026-07-16
- 来源边界：仅采用 `en.teikyo.com.cn` 的公开页面作为内容证据；页头、页脚中出现的站外入口只记录其存在，不用站外内容补证。

## 结论摘要

【页面事实】该站是湖南帝京新材料有限公司的英文企业站。全站主导航为 **Home、Product、Solutions、News、Service、About Us**；产品与新闻存在“栏目列表 → 详情”的三级层级，其余板块主要是单篇介绍页。首页还把公司简介、产品、行业方案/研发能力、新闻和联系方式汇总为品牌门户。[首页](https://en.teikyo.com.cn/)

【结构推断】这是一套以“展示企业能力、介绍涂料产品、收集询盘”为目标的 B2B 企业站：产品详情和留言表单承担转化，服务与关于我们承担信任背书，新闻承担品牌/行业传播。不过，解决方案全部为占位内容，两个产品分类为空，多个详情页存在复制或模板残留，因此当前公开内容不足以构成完整的产品选型或行业方案知识库。

## 4. 主要内容模型

### 4.1 产品

【页面事实】产品采用三级模型：**产品总索引 → 产品分类 → 产品详情**。分类实体至少有名称、分类 ID 和产品列表；详情实体包含名称、图片、浏览量、价格/市场价、货号、所属分类、数量、库存、描述、参数、上一篇/下一篇和留言表单。[产品总索引](https://en.teikyo.com.cn/cpzx1.html) · [汽车涂料列表](https://en.teikyo.com.cn/product/8/) · [产品详情](https://en.teikyo.com.cn/product/15.html)

【页面事实】公开数据中，汽车涂料和电子涂料各有 4 条详情；水性涂料和工业防腐涂料列表明确显示“当前无内容”。所有已抽查详情都保留 `0.0 元`、库存 `0`、空货号及“找不到对应参数”的模板字段。[汽车涂料](https://en.teikyo.com.cn/product/8/) · [电子涂料](https://en.teikyo.com.cn/product/7/) · [水性涂料](https://en.teikyo.com.cn/product/9/) · [工业防腐涂料](https://en.teikyo.com.cn/product/10/)

【结构推断】产品页更像“使用电商组件搭建的展示目录”，而不是可实际下单的商城：页面有价格、数量、库存，但价格和库存均为零，主要行动是 Message/留言。

### 4.2 解决方案

【页面事实】解决方案实体由标题、所属类别“行业解决方案”、发布时间、浏览量、Description、Information、正文和手机二维码组成。四个页面都显示三次 `Data is being collated...`，没有可用的行业流程、需求、产品组合、案例或技术参数。[汽车](https://en.teikyo.com.cn/intro/10.html) · [电子](https://en.teikyo.com.cn/intro/11.html) · [水性](https://en.teikyo.com.cn/intro/17.html) · [工业防腐](https://en.teikyo.com.cn/intro/18.html)

【结构推断】网站已经建立“按四条产品线一一对应四个行业方案”的分类骨架，但尚未填充方案正文。

### 4.3 新闻

【页面事实】新闻模型为 **新闻中心 → 新闻分类 → 新闻详情**。分类有名称、记录数和列表；详情含标题、精确到分钟的发布时间、正文和上一篇/下一篇。新闻中心共显示 4 条：Company News 3 条、Industry News 1 条。[新闻中心](https://en.teikyo.com.cn/news/1/) · [Company News](https://en.teikyo.com.cn/news/2/) · [Industry News](https://en.teikyo.com.cn/news/3/) · [详情示例](https://en.teikyo.com.cn/news/3.html)

【页面事实】可见内容只有 2014-07-10 和 2020-02-11 两个日期批次；截至访问日，官网未显示更晚的英文新闻。[新闻中心](https://en.teikyo.com.cn/news/1/)

【结构推断】新闻模块当前更像少量历史内容的存档，而不是持续更新的媒体中心。

### 4.4 服务

【页面事实】服务内容被拆成 Clean Room、R&D Center、Testing Center、Quality Management System 四篇单页，均采用“标题/类别/日期/正文/图片或二维码”的介绍页结构；正文分别讲生产洁净环境、研发团队与联合实验室、检测设备、质量部门与检验体系。[Clean Room](https://en.teikyo.com.cn/intro/9.html) · [R&D Center](https://en.teikyo.com.cn/intro/13.html) · [Testing Center](https://en.teikyo.com.cn/intro/14.html) · [Quality Management System](https://en.teikyo.com.cn/intro/15.html)

【结构推断】其信息架构不是按“售前/交付/售后”服务流程组织，而是按企业设施和管理能力组织，核心作用是制造与研发能力背书。

### 4.5 公司介绍与联系

【页面事实】About Us 包含公司简介、企业文化、领导寄语、合作、荣誉、联系我们和在线留言。公司简介是长篇企业沿革与能力自述；企业文化采用使命、愿景、精神等字段；荣誉为图片图集；联系页为结构化联系信息；留言页为表单。[Company Profile](https://en.teikyo.com.cn/intro/1.html) · [Corporate Culture](https://en.teikyo.com.cn/intro/2.html) · [Leadership Concern](https://en.teikyo.com.cn/intro/4.html) · [Honor](https://en.teikyo.com.cn/ryzz.html) · [Contact Us](https://en.teikyo.com.cn/intro/7.html) · [Message](https://en.teikyo.com.cn/zxly.html)

【结构推断】该模块同时承担三种任务：品牌叙事、信任证明、联系转化；但 Cooperation 主要依赖一张图片，检索和可访问性较弱。[Cooperation](https://en.teikyo.com.cn/intro/5.html)

## 6. 异常、重复、空内容与无法确认项

### 6.1 HTTP 可访问性与非导航链接

- 【测试事实】本研究枚举的 36 个核心内部 URL 在 2026-07-16 均返回 HTTP 200；未发现明确的内部 404/5xx 死链。该结论只覆盖上方站点地图，不代表隐藏页面或未来状态。
- 【页面事实】首页 HTML 中存在 `href="#"` 和 `javascript:void(0)` 一类无独立目标的交互占位；它们不是可抓取的内容页。[首页](https://en.teikyo.com.cn/)
- 【页面事实】页脚电话号码链接使用 `tel://0735-4521666`，而顶部热线使用 `tel:0086-0735-4521666`；【推断】前者的双斜线写法异常，跨设备兼容性无法仅从页面确认。[首页](https://en.teikyo.com.cn/)

### 6.2 空栏目与模板残留

- 【页面事实】[Waterborne Coatings](https://en.teikyo.com.cn/product/9/) 和 [Industrial Anticorrosive Coating](https://en.teikyo.com.cn/product/10/) 显示“当前无内容，请在网站后台添加数据记录”。
- 【页面事实】四个[解决方案页面](https://en.teikyo.com.cn/intro/10.html)均为 `Data is being collated...` 占位内容。
- 【页面事实】产品详情仍显示零价格、零库存、空货号、数量步进器和缺失参数提示；留言表单多个字段仍是中文。[产品详情示例](https://en.teikyo.com.cn/product/15.html)
- 【结构推断】这些现象说明英文站使用了通用产品/CMS 模板，但内容迁移或本地化没有完成。

### 6.3 重复或错配内容

- 【页面事实】[新闻 `/news/3.html`](https://en.teikyo.com.cn/news/3.html) 与 [新闻 `/news/4.html`](https://en.teikyo.com.cn/news/4.html) 的标题、日期和正文相同。
- 【页面事实】[Automobile interior coating](https://en.teikyo.com.cn/product/18.html) 与 [Automobile lamp coating](https://en.teikyo.com.cn/product/17.html) 都使用“vacuum plating UV coating”的同一段正文，该正文也出现在更匹配标题的 [PVD coatings](https://en.teikyo.com.cn/product/22.html)。
- 【结构推断】前者很可能是重复新闻记录，后者很可能是产品描述复制错位；官网没有给出修订说明，不能确认编辑过程。

### 6.4 路径、命名与本地化异常

- 【页面事实】路径体系混用语义不透明的数字 ID、拼音缩写和页面名：如 `/product/8/`、`/intro/17.html`、`/cpzx1.html`、`/ryzz.html`、`/zxly.html`。[Product Home](https://en.teikyo.com.cn/cpzx1.html) · [Honor](https://en.teikyo.com.cn/ryzz.html) · [Message](https://en.teikyo.com.cn/zxly.html)

## 7. 建议用于重构或迁移的标准化内容骨架（推断）

如果目标是依据现站内容重建信息架构，可把它规范为：

1. **首页**：品牌主张 → 核心产品线 → 行业方案 → 研发/质量能力 → 最新新闻 → 联系行动。
2. **产品**：产品线 → 产品系列/详情；详情字段保留应用对象、基材、涂层体系、性能、工艺、认证、资料下载和询盘，移除无实际交易用途的零价格/零库存字段。
3. **解决方案**：行业/场景 → 客户问题 → 推荐产品组合 → 工艺流程 → 性能指标 → 案例 → 联系技术团队。
4. **服务与能力**：研发、打样、检测、生产、质量、技术支持；比现有单纯设施介绍更接近客户旅程。
5. **新闻与资源**：公司新闻、行业洞察、技术资料；合并重复文章并修复日期。
6. **关于我们**：统一公司沿革、文化、管理层信息、合作伙伴、资质荣誉、联系方式；统一成立年份与地址后再发布。

这一节是基于现有内容的重构建议，不是官网当前已经实现的栏目。
